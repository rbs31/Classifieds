# Classifieds 📋

**North America Multi-Ethnic Classified Ads Platform**

A classified ads website that automatically detects your system language and prioritizes service providers who speak your language.

## ✨ Features

- 🌐 **Auto Language Detection** — Detects your browser/system language and shows matching service providers first
- 🏘️ **Multi-Ethnic Communities** — Dedicated listings for Chinese, Korean, Vietnamese, Spanish, Hindi, French, Portuguese, Russian, Arabic, and more
- 📂 **8 Categories** — Housing, Jobs, Vehicles, Services, For Sale, Community, Dating, Legal & Immigration
- 🔍 **Smart Search** — Search across all listings with location and category filters
- 📝 **Post Ads** — Create new listings with multi-language service tags
- 📱 **Responsive Design** — Works on desktop and mobile

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 🛠️ Tech Stack

- React 19 + Vite
- Pure CSS (no framework dependency)
- GitHub Pages deployment

## 🌐 Language Matching

The core differentiator: when a Chinese-speaking user visits the site, they see Chinese-speaking service providers first. When a Spanish-speaking user visits, they see Spanish-speaking providers first. This happens automatically based on browser language settings.
